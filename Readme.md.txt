# Sistema de Estoque - Loja Console App

Este projeto é um exercício que fiz do curso C# COMPLETO Programação Orientada a Objetos + Projetos da Udemy que desenvolvi algumas opções extras além do requisitado. Ele é uma aplicação de console feita em C#, utilizando Programação Orientada a Objetos (POO), que simula o gerenciamento de estoque de produtos de uma loja.

## Funcionalidades

- Leitura de dados de um produto (nome, preço, quantidade)
- Adição ou remoção de itens no estoque
- Cálculo do valor total em estoque
- Validação de entradas com TryParse para evitar erros

## Tecnologias usadas

- C#
- .NET
- Programação Orientada a Objetos (POO)
- Validação com TryParse
